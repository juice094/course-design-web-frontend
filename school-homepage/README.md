## 📐 组件复用方式

纯静态 HTML 无服务器端包含（SSI/PHP），组件复用采用以下策略：

1. **开发阶段**：修改 `components/*.html`，然后复制粘贴到各页面
2. **后续可升级**：引入前端构建工具（Vite/Webpack）或服务器模板引擎
3. **当前实践**：`index.html` 已作为标准模板，其他子页面复制其 `<head>` 和首尾结构
